# Wireshark Network Analysis Projects
Network Analysis on Captured pcap files to understand basic to advanced scanning and malicious activities

## Overview
This project analyzes:
1. Network Reconnaissance Detection
2. ARP Spoofing or MIMT attack
3. DNS Analysis & Tunneling Detection
4. HTTP vs HTTPS traffic Analysis
5. TCP analysis and RST Attacks
6. TCMP flood detections

## Objectives
- Oversight controlled captured .pcap file 
- Identify indicators in `.pcap` files
- Build detection filters using Wireshark display filters
- Document findings with steps and annotated screenshots

## Lab Setup
- Virtual Machines: Kali Linux, Parrot OS
- Network: VirtualBox Bridged Network, Local to Local Port Scanning
- Tools: Wireshark, Nmap, Scapy, Netcat, Python
- PCAP Files: Provided in each project directory
